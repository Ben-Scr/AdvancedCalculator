# Advanced WPF Calculator
A advanced calculator for windows.

## Calculation Examples
- 2 + 5 * 2 = 12
- 5 * (10 + 5) = 75
- 20 + 100 / 2 * 4^2 = 820
- 5 + (-2) - (-12) = 15

## Current features
- Functions such as sin, cos and tan
- Constant Variables like π and E
- Advanced calculations such as (sin(π ÷ 6)) ^ 2 + (cos(π ÷ 6)) ^ 2 = 1
