# Advanced WPF Calculator
An advanced calculator for windows.

## Calculation Examples
- 20 + 100 / 2 * 4^2 = 820
- 5 + (-2) - (-12) = 15
- (sin(π ÷ 6)) ^ 2 + (cos(π ÷ 6)) ^ 2 = 1
- cos(0) = 1
- √(2 ^ 2 + 3 ^ 2) = 3.60555127546399

## Implemented functions
- pow(value1, value2)
- sin(value)
- cos(value)
- tan(value)
- log(value)
- abs(value)
- atan(value)
- atan2(value1, value2)
- max(value1, value2, ...)
- min(value1, value2, ...)
- clamp(value, min, max)
- random(min, max)
- random()
- toHex(value1, value2, ...)
- quit()
